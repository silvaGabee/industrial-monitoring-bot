# Política de Privacidade — Bot Telegram

Política de privacidade do Bot de Monitoramento Industrial via Telegram.

**Última atualização:** junho de 2026

## Visão Geral

Este bot coleta e processa informações operacionais de equipamentos industriais conectados com a finalidade de monitoramento, supervisão e controle remoto.

## Dados Coletados

As informações processadas podem incluir:

* Status dos equipamentos
* Contadores de produção
* Alarmes e eventos operacionais
* Comandos enviados pelos usuários autorizados via Telegram

## Uso das Informações

Os dados são utilizados exclusivamente para:

* Monitoramento em tempo real
* Controle remoto de equipamentos
* Geração de notificações e alertas
* Exibição de informações operacionais

Nenhuma informação é vendida, compartilhada ou utilizada para fins comerciais.

## Segurança

O acesso às funcionalidades do bot é restrito a usuários autorizados. Os usuários são responsáveis por manter a segurança de suas credenciais e dispositivos de acesso.

## Contato

Para dúvidas relacionadas à privacidade ou ao tratamento das informações processadas por este bot, entre em contato com o administrador responsável.

## Alterações nesta Política

Esta política poderá ser atualizada periodicamente. A versão mais recente estará sempre disponível nesta página.
